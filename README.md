# Bilibili Clone - 哔哩哔哩复刻版

一个高度仿真的哔哩哔哩视频网站复刻版本，包含完整的视频浏览、播放和评论功能。

## ✨ 项目特色

- 🎨 **高度还原B站界面** - 精确复刻B站的视觉风格和交互体验
- 🎬 **真实可播放视频** - 所有视频均可正常播放，无虚假链接
- 📱 **响应式设计** - 完美适配电脑和移动设备
- 🎯 **完整功能** - 视频播放、评论、关注、搜索一应俱全
- 🌈 **多分区展示** - 热门推荐、动画区、音乐区等
- 💬 **实时评论** - 支持发表和浏览评论

## 🎮 功能说明

### 核心功能

| 功能 | 描述 |
|------|------|
| 🏠 首页推荐 | 精选热门视频推荐 |
| 🔍 视频搜索 | 支持按标题和UP主搜索 |
| 📂 分区浏览 | 动画、音乐等多个分区 |
| ▶️ 视频播放 | 内置播放器，支持控制 |
| 💬 评论区 | 支持浏览和发表评论 |
| ❤️ 关注UP主 | 一键关注喜欢的主播 |
| 🏷️ 分类筛选 | 按类别快速筛选视频 |

### 操作说明

**浏览视频**
- 点击任意视频卡片即可播放
- 支持点击封面或标题打开播放页面

**搜索功能**
- 在顶部搜索框输入关键词
- 按回车或点击搜索按钮
- 支持搜索视频标题和UP主名称

**分类筛选**
- 点击分类标签可筛选对应分区视频
- "全部分类"显示所有视频

**评论互动**
- 播放页面下方可发表评论
- 点击"发布"按钮提交评论
- 可查看其他用户的评论

## 📝 数据说明

### 视频数据格式

```javascript
{
    id: 1,                          // 视频ID
    title: "视频标题",              // 视频名称
    cover: "封面图片URL",           // 封面链接
    videoUrl: "视频源地址",         // 视频链接（真实可播放）
    duration: "03:45",             // 视频时长
    views: "125.3万",              // 播放量
    likes: "8.2万",                // 点赞数
    date: "2024-01-15",            // 发布日期
    upName: "UP主名称",            // UP主名称
    upAvatar: "头像URL",           // UP主头像
    upFollowers: "128万",          // 粉丝数
    description: "视频简介",       // 视频描述
    category: "life"               // 分类
}
```

## 🔧 自定义配置

```javascript

    recommended: [

            title: "新视频标题",
            cover: "封面图片URL",
            videoUrl: "视频URL",
            // ... 其他字段
        }
    ]
};
```

### 修改主题色

在 `style.css` 中修改 `#FB7299` 为自定义颜色。
